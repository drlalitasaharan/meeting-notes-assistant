## Demo positioning

Meeting Notes Assistant converts meeting recordings into structured business notes:

- Purpose
- Outcome
- Risks
- Next Steps
- Key Points
- Decisions
- Action Items

## Recommended demo flow

1. Open with the problem: teams lose time turning meetings into usable follow-up notes.
2. Show the upload flow.
3. Explain that processing creates structured notes.
4. Show the external-ready markdown output.
5. Highlight decisions and action items.
6. Explain fallback and reliability plan.
7. Close with the pilot next step.

## Demo rule

Use a short, clean live demo file for speed and predictability.

Use the validated 10-minute artifact as proof of stronger capability, not necessarily as the live upload during the client call.
