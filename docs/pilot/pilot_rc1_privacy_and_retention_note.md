# Pilot RC1 Privacy and Retention Note

## User-facing pilot privacy note

Your uploaded meeting files are used only to generate notes for this pilot.
We do not sell your data.
Pilot files can be deleted on request.
Please avoid uploading highly confidential, legal, medical, or regulated recordings during the first pilot.

## Internal checklist

Confirm before onboarding first users:

- audio storage location
- transcript storage location
- generated notes storage location
- retention period for uploaded audio
- retention period for transcripts
- retention period for generated notes
- who can access pilot data
- deletion request process
- whether logs contain transcript snippets
- whether pilot data is used for product improvement

## Recommended pilot retention default

Delete uploaded audio after the pilot review period unless the user agrees to keep it for debugging.
Keep generated notes only as long as needed for the pilot.
Delete user data on request.
Avoid storing unnecessary transcript snippets in logs.

## Sensitive content warning

Pilot users should avoid uploading:

- legal recordings
- medical recordings
- regulated financial recordings
- confidential HR recordings
- highly sensitive customer data
- recordings without participant permission
